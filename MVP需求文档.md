## 三、技术架构

### 3.1 技术选型
| 层级 | 技术 | 说明 |
|---|---|---|
| 前端 | 原生 HTML/CSS/JS | 轻量，无框架依赖 |
| 后端 | Cloudflare Workers | Serverless，全球边缘节点 |
| AI 服务 | Remove.bg API | 专业抠图模型 |
| 部署平台 | Cloudflare | 免费套餐够用 |
| 存储 | 无 | 图片不落地存储 |

### 3.2 数据流
```
用户上传图片
    ↓
Cloudflare Worker 接收（内存中）
    ↓
转发至 Remove.bg API
    ↓
接收处理结果（PNG）
    ↓
直接返回给用户浏览器
    ↓
用户本地下载（图片不存储）
```

### 3.3 API 接口设计
```
GET  /              → 返回前端 HTML 页面
POST /api/remove-bg → 接收图片，调用 Remove.bg，返回 PNG
```

**POST /api/remove-bg**
- 请求：`multipart/form-data`，字段名 `image_file`
- 响应成功：`image/png`，透明底 PNG 文件
- 响应失败：`application/json`，`{ "error": "错误信息" }`
